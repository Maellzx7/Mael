Programação Web

A Programação Web ensina a criar sites e aplicativos para a internet. Nessa matéria, os alunos aprendem linguagens como HTML, CSS e JavaScript para o front-end e linguagens como PHP e Python para o back-end. O curso aborda desde o funcionamento básico de navegadores e servidores até a criação de sistemas web seguros e interativos.
